Fantasy PunchOut Game Document

Game Design Document

A game design document for a game that is a mixture of a traditional RPG where the battles are instead turned into a first person PunchOut-style combat. It is a concept that I have been working on for a while, I imagine that it would be a game where you traverse the world with walking into enemies would start combat. The combat would be sword and shield based, but reminiscent of PunchOut or King's Feild in a stationary location. The visuals would be a classic 16-bit style on the overworld with a PSX style for the punchout combat. I am trying to make smaller games for the overworld and the sword and sheild based combat to start off with.
